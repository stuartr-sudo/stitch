                          DATABASE SCHEMA
════════════════════════════════════════════════════════════════════════════════

Table: generated_audio

Columns:
  id              uuid primary key
  user_id         uuid (RLS, references auth.users)
  title           text
  prompt          text
  negative_prompt text
  model           text (beatoven/music-generation, etc)
  audio_url       text (Supabase storage URL)
  duration_seconds float
  refinement      integer
  creativity      float
  seed            integer
  created_at      timestamptz
  updated_at      timestamptz

Security:
  ✅ RLS Enabled
  ✅ Row Level Security Policies (3)
  ✅ User Isolation
  ✅ Auto-update timestamp trigger

Performance:
  ✅ Index on user_id
  ✅ Index on model
  ✅ Optimized for queries
