# Video Style Frameworks - Stitch Studios

Reference document for all video format types, their structures, and pipeline configuration.

---

## Story Styles

These are narrative-driven, continuous flow formats. Frame-chaining is ON. Transitions are crossfades. TTS is generated as a single continuous file (not per-scene). Pacing is slower, with longer scenes.

### Personal Journey
*"How I went from [A] to [B]"*
*"What [experience] taught me about [lesson]"*
*"The moment everything changed for me"*

| Property | Value |
|---|---|
| Duration | 60s or 90s |
| Scenes | 5 (60s) / 7 (90s) |
| Scene duration | 10-14s each |
| Frame chain | ON |
| Transitions | Crossfade (1.5s) |
| TTS mode | Single continuous file |
| TTS pacing | Slow, reflective, conversational |
| Text overlays | None (let visuals breathe) |
| Music | Ambient/emotional, 20% volume, fade in on scene 1, fade out on final scene |

**Scene structure:**
1. **Setup** (10-12s) - Establish the "before" state. Relatable situation.
2. **Inciting moment** (10-12s) - The thing that changed. Tension builds.
3. **Struggle/middle** (10-14s) - The difficulty, the process, the messy middle.
4. **Turning point** (10-12s) - The breakthrough or realization.
5. **Resolution** (10-12s) - The "after" state. Payoff.
6. **Reflection** (10-12s) - *90s only.* What it means now. Broader lesson.
7. **Call to meaning** (8-10s) - *90s only.* Universal takeaway. Lingering emotion.

### Origin Story
*"How [brand/person/thing] started in a [humble place]"*
*"The real story behind [X]"*
*"From [nothing] to [something] - the [X] story"*

| Property | Value |
|---|---|
| Duration | 60s or 90s |
| Scenes | 5 (60s) / 7 (90s) |
| Scene duration | 10-14s each |
| Frame chain | ON |
| Transitions | Crossfade (1.5s) |
| TTS mode | Single continuous file |
| TTS pacing | Storytelling tone, builds energy over time |
| Text overlays | Optional date/year stamp on scene 1 |
| Music | Inspirational build, starts quiet, peaks at scene 4-5 |

**Scene structure:**
1. **The humble beginning** (10-12s) - Where it all started. Small, scrappy.
2. **The spark** (10-12s) - The idea, the gap, the moment of clarity.
3. **Early struggles** (10-14s) - What went wrong. Rejection, failure, doubt.
4. **The break** (10-12s) - First success. Validation. Momentum.
5. **Where they are now** (10-12s) - The payoff. Scale. Impact.
6. **Behind the scenes** (10-12s) - *90s only.* What people don't see.
7. **Legacy/future** (8-10s) - *90s only.* What's next. Lasting impact.

### Mini Documentary
*"The untold story of [X]"*
*"Inside [X]: what really happens"*
*"The hidden world of [X]"*

| Property | Value |
|---|---|
| Duration | 60s or 90s |
| Scenes | 5 (60s) / 7 (90s) |
| Scene duration | 10-14s each |
| Frame chain | ON |
| Transitions | Crossfade (1s) |
| TTS mode | Single continuous file |
| TTS pacing | Authoritative, measured, documentary narrator |
| Text overlays | Optional location/context label on scene 1 |
| Music | Cinematic underscore, 15-20% volume |

**Scene structure:**
1. **Cold open** (8-10s) - Drop the viewer into the middle of something compelling.
2. **Context** (10-12s) - Pull back. Explain what this is and why it matters.
3. **Deep dive** (12-14s) - The core of the story. Detail, specifics.
4. **Complication** (10-12s) - The twist, the problem, the tension.
5. **Resolution/insight** (10-12s) - What we learn. The takeaway.
6. **Wider implications** (10-12s) - *90s only.* Why this matters beyond this story.
7. **Closing thought** (8-10s) - *90s only.* Lingering question or statement.

*"What it's actually like to [do X]"*

| Property | Value |
|---|---|
| Duration | 45s or 60s |
| Scenes | 4 (45s) / 5 (60s) |
| Scene duration | 10-12s each |
| Frame chain | ON |
| Transitions | Crossfade (1s) |
| TTS mode | Single continuous file |
| TTS pacing | Casual, warm, first-person feel |
| Text overlays | Time stamps ("6:00 AM", "2:30 PM") on each scene |
| Music | Lo-fi/chill, 20% volume, continuous |

2. **Core work** (10-12s) - The main thing they do. The interesting part.
3. **Unexpected moment** (10-12s) - Something surprising or behind-the-scenes.
4. **Wind down** (10-12s) - End of day. Reflection.
5. **Why they love it** (8-10s) - *60s only.* The emotional hook.

## Fast-Paced / List Styles

These are punchy, high-energy formats. Frame-chaining is OFF (each scene is independent). Transitions are hard cuts. TTS is generated per-scene. Text overlays are required.

### Top X / Countdown
*"Top 5 [X] you've never heard of"*
*"3 [X] that will change how you [Y]"*
*"Ranking every [X] from worst to best"*
*"The only 3 [X] you'll ever need"*
*"5 [X] under $[Y] that are actually worth it"*

| Property | Value |
|---|---|
| Duration | 30s, 45s, or 60s |
| Scenes | 3 (30s) / 4-5 (45s) / 5-6 (60s) - one per item + intro |
| Scene duration | 5-8s per item, 3-4s intro |
| Frame chain | OFF |
| Transitions | Hard cut |
| TTS mode | Per-scene |
| TTS pacing | Fast, punchy, energetic |
| Text overlays | REQUIRED - large number (5, 4, 3, 2, 1) + item name |
| Music | Upbeat/energetic, 20% volume |

**Scene structure (60s, Top 5 example):**
1. **Hook** (4-5s) - "Here are 5 [X] that [hook]." Text: title card.
2. **#5** (8-10s) - Lowest ranked. Quick intro + why. Text: "5" + name.
3. **#4** (8-10s) - Text: "4" + name.
4. **#3** (8-10s) - Text: "3" + name.
5. **#2** (8-10s) - Build anticipation. Text: "2" + name.
6. **#1** (10-12s) - Longest scene. Best item. Text: "1" + name.

**Variant - count UP (1 to 5):** Same structure, reverse order. Works for "worst to best."

### Everything You Need to Know
*"Everything you need to know about [X] in 60 seconds"*
*"[X] explained in under a minute"*
*"30 seconds to understand [X]"*
*"Learn [X] faster than you can make coffee"*

| Property | Value |
|---|---|
| Duration | 30s or 60s |
| Scenes | 3 (30s) / 5 (60s) |
| Scene duration | 8-12s each |
| Frame chain | OFF |
| Transitions | Hard cut |
| TTS mode | Per-scene |
| TTS pacing | Fast but clear, no pauses, information-dense |
| Text overlays | REQUIRED - keyword/topic label per scene |
| Music | Ticking/urgent, 15% volume |

**Scene structure (60s):**
1. **What is it?** (8-10s) - Definition. Core concept. Text: "[X]" title.
2. **Why does it matter?** (10-12s) - Relevance. Why you should care. Text: keyword.
3. **How does it work?** (12-14s) - Mechanics. The substance. Text: keyword.
4. **Key fact/stat** (10-12s) - One surprising number or detail. Text: the stat.
5. **Bottom line** (6-8s) - One sentence takeaway. Text: takeaway phrase.

### Myth Busting
*"3 myths about [X] that are completely wrong"*
*"You've been doing [X] wrong your whole life"*
*"Stop doing [X] - here's why"*
*"The truth about [X] nobody talks about"*

| Property | Value |
|---|---|
| Duration | 30s, 45s, or 60s |
| Scenes | 3 (30s) / 4 (45s) / 5 (60s) |
| Scene duration | 8-12s each |
| Frame chain | OFF |
| Transitions | Hard cut |
| TTS mode | Per-scene |
| TTS pacing | Confident, slightly provocative, debunking energy |
| Text overlays | REQUIRED - "MYTH" (red) then "TRUTH" (green) per item |
| Music | Dramatic/revelatory, 15% volume |

**Scene structure (60s, 3 myths):**
1. **Hook** (5-6s) - "You've been told [X]. That's wrong." Text: title.
2. **Myth 1 + Truth** (12-14s) - State myth, pause, reveal truth. Text: MYTH / TRUTH.
3. **Myth 2 + Truth** (12-14s) - Same pattern. Text: MYTH / TRUTH.
4. **Myth 3 + Truth** (12-14s) - Same pattern. Text: MYTH / TRUTH.
5. **What to do instead** (8-10s) - Actionable takeaway. Text: takeaway.

### Comparison / Versus
*"[X] vs [Y] - which one actually wins?"*
*"Cheap vs expensive [X] - is it worth it?"*
*"How [X] looked in [year] vs now"*

| Property | Value |
|---|---|
| Duration | 30s or 45s |
| Scenes | 3 (30s) / 4 (45s) |
| Scene duration | 8-12s each |
| Frame chain | OFF |
| Transitions | Hard cut or split-screen wipe |
| TTS mode | Per-scene |
| TTS pacing | Analytical, fair, then decisive at the verdict |
| Text overlays | REQUIRED - "[X] vs [Y]" title, winner label |
| Music | Competitive/sporty, 15% volume |

**Scene structure (45s):**
1. **The matchup** (5-6s) - Introduce both. Text: "[X] vs [Y]".
2. **Contender A** (10-12s) - Pros, cons, key detail. Text: name + key stat.
3. **Contender B** (10-12s) - Same treatment. Text: name + key stat.
4. **The verdict** (8-10s) - Clear winner with reasoning. Text: "WINNER: [X]".

## Pipeline Config Summary

Quick reference for how each style category maps to pipeline settings.

| Style Category | Frame Chain | TTS Mode | Transitions | Text Overlays | Music Vol |
|---|---|---|---|---|---|
| Personal Journey | ON | Single file | Crossfade 1.5s | None | 20% |
| Origin Story | ON | Single file | Crossfade 1.5s | Optional date | 20-25% |
| Mini Documentary | ON | Single file | Crossfade 1s | Optional label | 15-20% |
| Day in the Life | ON | Single file | Crossfade 1s | Time stamps | 20% |
| Before & After | ON | Single file | Wipe at reveal | Before/After | 20% |
| Explainer Story | ON | Single file | Crossfade 1s | Minimal | 15% |
| Emotional/Tribute | ON | Single file | Slow crossfade 2s | None | 25% |
| Top X / Countdown | OFF | Per-scene | Hard cut | Numbers + names | 20% |
| Everything in 60s | OFF | Per-scene | Hard cut | Keywords | 15% |
| Myth Busting | OFF | Per-scene | Hard cut | Myth/Truth | 15% |
| Comparison | OFF | Per-scene | Hard cut / wipe | Names + verdict | 15% |
| Did You Know | OFF | Per-scene | Hard cut | Fact numbers | 15% |
| Challenge | ON | Single file | Crossfade 1s | Day markers | 20% |
| History/Timeline | OFF | Per-scene | Hard cut / wipe | Years | 15% |
| Hot Take | OFF | Per-scene | Hard cut | Bold statements | 15% |
| How It Works | ON | Single file | Crossfade 1s | Optional labels | 10-15% |

### Key Decision Tree

```
Is the video a continuous narrative?
├── YES → Frame chain ON, single TTS file, crossfade transitions
│   └── Does it have time markers? (Day in Life, Challenge, History)
│       ├── YES → Add time/date text overlays
│       └── NO → Minimal or no text overlays
└── NO → Frame chain OFF, per-scene TTS, hard cuts
    └── Text overlays are REQUIRED (numbers, labels, keywords)
```
